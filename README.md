# Superstore Sales Analytics Dashboard

This project is a Power BI dashboard analyzing retail sales data.

## Dashboard Features
- Total Sales, Profit and Orders KPIs
- Profit by State
- Sales by Category
- Top Products by Sales
- Orders Trend Over Time
- Region and State filters

## Tools Used
- Power BI
- DAX
- Data Visualization
