# Superstore Sales Analytics Dashboard

## 📊 Project Overview

This project analyzes retail sales data using **Microsoft Power BI**.
The dashboard provides insights into sales performance, profit trends, product demand, and regional distribution.

## 🎯 Business Objective

To identify sales patterns, top-performing products, and regional performance to support data-driven business decisions.

## 📈 Dashboard Features

* Total Sales KPI
* Total Profit KPI
* Total Orders KPI
* Profit by State
* Sales by Category
* Top Products by Sales
* Orders Trend Over Time
* Interactive filters (Category, Region, State)

## 🛠 Tools & Technologies

* Microsoft Power BI
* DAX (Data Analysis Expressions)
* Data Visualization
* Superstore Dataset

## 📁 Files Included

* `Superstore_Dashboard.pbix` – Power BI dashboard
* `superstore.csv` – dataset used
* `dashboard.png` – dashboard preview image

## 🚀 How to Use

1. Download the `.pbix` file
2. Open it using **Power BI Desktop**
3. Use slicers to explore the dashboard interactively
